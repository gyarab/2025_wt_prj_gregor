# FightLog – MMA & BJJ Training Tracker

FightLog je webová aplikace vytvořená v Django frameworku, která slouží jako osobní tréninkový deník pro MMA a BJJ zápasníky. Uživatel si po každém tréninku může zaznamenat typ tréninku (MMA, BJJ, sparing, technika), datum, délku tréninku a vlastní poznámky. Součástí aplikace je také evidence výsledků ze sparingů ve formátu win–lose–draw.

Systém zároveň sleduje statistiky submisí – uživatel uvidí přehled, na jaké techniky nejčastěji vyhrává a na jaké naopak prohrává. Data jsou ukládána do databáze a zobrazována přehlednou formou, například pomocí grafů nebo procentuálního rozložení úspěšnosti.
